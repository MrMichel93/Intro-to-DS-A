## ğŸ“ Assessment Types

### 1. Module Checkpoint Quizzes (11 total)

**Purpose:** Assess understanding of individual module concepts  
**When to Take:** After completing each module  
**Time Limit:** 45 minutes each  
**Points:** 100 points each

Each checkpoint quiz includes:
- **10 multiple choice questions** (40 points) - Test conceptual understanding
- **2-3 code analysis questions** (30 points) - Analyze code and determine complexity
- **2 short answer questions** (30 points) - Explain concepts and apply knowledge

**Modules covered:**
- Module 0: Foundations (Big O, problem-solving)
- Module 1: Arrays
- Module 2: Linked Lists
- Module 3: Stacks
- Module 4: Queues
- Module 5: Hash Tables
- Module 6: Trees
- Module 7: Binary Search Trees
- Module 8: Graphs
- Module 9: Searching Algorithms
- Module 10: Sorting Algorithms

### 2. Midterm Challenge

**File:** `midterm_challenge.md`  
**When to Complete:** After Module 5 (Hash Tables)  
**Time Estimate:** 3-4 hours  
**Points:** 100 points

**Project:** Build a Contact Management System

Demonstrates mastery of:
- Arrays and dynamic data structures
- Linked lists for ordering
- Stacks for undo/redo
- Queues for bulk operations
- Hash tables for fast lookups

**Key Features:**
- Core contact CRUD operations with O(1) lookups
- Undo/redo functionality using stacks
- Bulk operations queue processing
- Recently accessed contacts tracking
- Multiple search/filter capabilities

### 3. Final Project

**File:** `final_project.md`  
**When to Complete:** After Module 10 (course completion)  
**Time Estimate:** 8-12 hours over 2-3 weeks  
**Points:** 200 points

**Project:** Build a Task Management System

Demonstrates mastery of:
- All data structures covered in the course
- Multiple sorting and searching algorithms
- Graph algorithms (dependencies, topological sort)
- Real-world system design
- Efficient algorithm implementation

**Key Features:**
- Comprehensive task management (CRUD operations)
- Task dependencies using graphs
- Priority-based scheduling with heaps
- Advanced search and filtering
- Multiple sorting options
- Undo/redo system
- Analytics and reporting
- Choice of advanced features (subtasks, recurring tasks, etc.)

**Requirements:**
- Must use at least 5 different data structures
- Must implement at least 3 algorithms from scratch
- Comprehensive testing required
- Full documentation expected

## â“ Common Questions

**Q: Can I use external libraries?**  
A: For checkpoints, no. For projects, only Python standard library unless otherwise specified.

**Q: What if I don't finish a checkpoint in 45 minutes?**  
A: Note where you stopped, then review and study those areas. Checkpoints test your current understanding.

**Q: Can I reuse code from the practice problems?**  
A: Yes, but understand it thoroughly. You may need to modify it significantly.

**Q: How much time should I spend on the final project?**  
A: Plan for 8-12 hours total, spread over 2-3 weeks. Don't underestimate documentation and testing time.

**Q: What if I get stuck on a project?**  
A: Review relevant modules, check the hints section, draw diagrams, or ask for help during office hours.

**Q: Are the answer keys available to students?**  
A: Instructor decides. We recommend students check answers only after attempting the quiz.
